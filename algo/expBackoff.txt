## 指数退避算法

1. 参考：https://blog.csdn.net/chdhust/article/details/48524511 
2. 指数退避算法是用于重试时的一种策略。在写代码中遇到一种可能失败但是需要重试(重试需保证幂等)的场景时，可以使用指数退避算法。比如在进行网络访问时，第一次访问失败，延迟一定时间再重试，延迟时间再(0,1)中随机选择，如果第二次再访问失败，延迟时间在(0, 1, 2, 3)中随机选择，这样最大延迟时间成指数增长，可以降低网络负载过高时的访问冲突概率，为了避免重试等待时间过长，可以设置最大等待时间，使指数增长不会超过最大时间
3. 实例：

```
MAX = 10
while(true) {
	int exp = 1;
	int rs = doOp();
	if (rs == Succ) {
		return;
	}
	int delay = Math.random(0, exp);
	exp *= 2;
	sleep(delay);
}
```
